# my-connector2

My Commercetools Connect Connector

## Overview

This is a Commercetools Connect connector that provides integration capabilities.

## Configuration

The connector requires the following environment variables:

- `CTP_PROJECT_KEY`: Your Commercetools project key
- `CTP_CLIENT_ID`: Your Commercetools client ID
- `CTP_CLIENT_SECRET`: Your Commercetools client secret

## Development

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test
```

## Deployment

The connector is automatically deployed via GitHub Actions when changes are pushed to the `main` branch.
